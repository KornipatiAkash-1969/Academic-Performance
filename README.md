# Academic Performance System

A complete full-stack web application developed for managing student academic performance, subjects, marks, assessments, and academic coordination.

---

# 🚀 Tech Stack

## Frontend
- ReactJS
- React Router DOM
- Axios
- CSS

## Backend
- Node.js
- Express.js
- SQLite

## Database
- SQLite Database

---

# 📌 Features

## 👨‍🏫 Teacher Portal
- Teacher Login
- Add Students
- View Students List
- Add Subjects
- Add Marks
- Manage Academic Records

## 👨‍🎓 Student Portal
- Student Login
- View Subjects
- View Marks
- View Grades
- View Academic Performance

## 👨‍💼 Coordinator Portal
- Coordinator Login
- Create Teacher Accounts
- Send Notes
- Academic Coordination Dashboard

# ▶️ How To Run Project

---

## 🔹 Backend Setup

### Step 1

Open terminal:

```bash
cd backend
```

### Step 2

Install dependencies:

```bash
npm install
```

### Step 3

Run backend server:

```bash
npm run dev
```

### Backend Running On

```bash
http://localhost:5000
```

---

## 🔹 Frontend Setup

### Step 1

Open new terminal:

```bash
cd frontend
```

### Step 2

Install dependencies:

```bash
npm install
```

### Step 3

Start React application:

```bash
npm start
```

### Frontend Running On

```bash
http://localhost:3000
```
